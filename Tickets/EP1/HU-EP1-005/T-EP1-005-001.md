# T-EP1-005-001: Create photo management API endpoints

## Title
Create photo management API endpoints

## Description

Implement DELETE, PUT endpoints for photo deletion, reordering, and primary photo management.

**Purpose:**
- Enable photo deletion with file cleanup
- Allow photo reordering in galleries
- Support primary photo designation changes

**Details:**
- DELETE /api/products/{id}/photos/{photoId}
- PUT /api/products/{id}/photos/{photoId}/order
- PUT /api/products/{id}/photos/{photoId}/primary
- File deletion from storage
- DisplayOrder updates
- IsPrimary logic (only one primary per product)

## Acceptance Criteria

- [ ] Photo deletion endpoint working
- [ ] Photo reordering implemented
- [ ] Primary photo designation working
- [ ] File cleanup on deletion
- [ ] Business rules enforced
