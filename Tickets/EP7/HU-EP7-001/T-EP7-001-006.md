# T-EP7-001-006: Implement login page with Metronic components

## Purpose

Create the login page using Metronic React components to provide a professional, responsive user interface that works on both desktop and mobile devices. Implement form validation, error handling, and loading states.

## Details

### Login Page Components
- Create LoginPage component using Metronic's form.tsx and input.tsx components
- Implement responsive design that works on mobile (operator priority) and desktop
- Use Metronic's card.tsx for layout container
- Include loading states and error messages using Metronic components

### Form Implementation
- Username field (text input) - required validation
- Password field (password input with show/hide toggle) - required validation
- Login button with loading state
- "Forgot password" link (placeholder for future implementation)

### User Experience
- Auto-focus on username field
- Enter key submits form
- Loading indicator during authentication
- Clear error messages for validation failures
- Remember last username on mobile (optional enhancement)

### Error Handling UI
- Display authentication errors (invalid credentials, inactive user, etc.)
- Form validation errors (required fields)
- Network error handling with retry option

### Metronic Components to Use
- `form.tsx` - Form container with react-hook-form integration
- `input.tsx` - Username and password inputs
- `button.tsx` - Login button with loading state
- `card.tsx` - Page container
- `alert.tsx` or `toast` - Error messages

## Acceptance Criteria

- [ ] Login page is created using Metronic components
- [ ] Form includes username and password fields with validation
- [ ] Responsive design works on mobile and desktop
- [ ] Loading states are properly displayed during authentication
- [ ] Error messages are user-friendly and clear
- [ ] Form can be submitted with Enter key
- [ ] Metronic form.tsx component is properly integrated
- [ ] Page follows Metronic design system
