# T-EP1-002-002: Create product creation form page

## Title
Create product creation form page

## Description

Create React page with product creation form using Metronic components (form, input, select, button) with validation, collection selection, and proper user feedback.

**Purpose:**
- Provide user-friendly form for manual product creation
- Include real-time validation and error feedback
- Support collection assignment with dropdown
- Show success confirmation after creation

**Details:**
- Use Metronic `form` component with React Hook Form integration
- Form fields: SKU, Name, Description, Price, Collection (dropdown)
- Real-time validation with error messages
- Collection dropdown populated from API
- SKU uniqueness validation (async check)
- Price input with number formatting and validation
- Success feedback with option to create another or navigate
- Proper loading states during submission

**References:**
- User Story: `Documentos/Historias/HU-EP1-002.md` (form requirements)

- Form validation: React Hook Form + Zod integration
