# HU-EP1-005: Gestionar fotos de producto

## Formato estándar

Como **administrador**, quiero **gestionar las fotos de productos (eliminar, reordenar, marcar principal)** para **mantener el catálogo de fotos actualizado y optimizado**.

---

## Descripción

Permite eliminar fotos, cambiar el orden de visualización y marcar una foto como principal.

---

## Criterios de Aceptación

### Escenario 1: Eliminar foto
**Dado que** un producto tiene múltiples fotos  
**Cuando** elimino una foto  
**Entonces** el sistema elimina el archivo y el registro  
**Y** la foto desaparece de la galería

### Escenario 2: Reordenar fotos
**Dado que** un producto tiene fotos en orden A, B, C  
**Cuando** cambio el orden a B, C, A  
**Entonces** el sistema actualiza `DisplayOrder`  
**Y** las fotos se muestran en el nuevo orden

### Escenario 3: Marcar foto principal
**Dado que** un producto tiene múltiples fotos sin principal  
**Cuando** marco una foto como principal  
**Entonces** `IsPrimary = true` para esa foto  
**Y** las demás quedan con `IsPrimary = false`

---

## Tareas

1. Backend: Endpoints `DELETE /api/products/{id}/photos/{photoId}`, `PUT /api/products/{id}/photos/{photoId}/order`, `PUT /api/products/{id}/photos/{photoId}/primary`
2. Frontend: Interfaz de gestión con drag & drop para reordenar
