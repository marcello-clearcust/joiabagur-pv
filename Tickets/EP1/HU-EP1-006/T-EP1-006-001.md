# T-EP1-006-001: Create product catalog API endpoint

## Title
Create product catalog API endpoint

## Description

Implement GET /api/products endpoint with pagination for administrators and operators to browse the product catalog.

**Purpose:**
- Provide paginated product listing
- Include primary photo information
- Support catalog browsing for all authenticated users

**Details:**
- GET /api/products with pagination parameters
- Include primary photos in response
- Default page size 50, configurable
- Sort by name or creation date
- Return ProductDto with essential information

## Acceptance Criteria

- [ ] Paginated product listing working
- [ ] Primary photo included in responses
- [ ] Proper pagination metadata
- [ ] Authentication required
- [ ] Performance optimized for large catalogs

## Priority
High (catalog browsing functionality)

## Estimation
2 SP
