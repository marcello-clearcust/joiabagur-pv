# HU-EP1-002: Crear producto manualmente

## Formato estándar

Como **administrador**, quiero **crear productos manualmente en el sistema** para **agregar productos individuales al catálogo sin necesidad de importar desde Excel**.

---

## Descripción

El sistema debe permitir crear productos individuales con SKU, nombre, descripción, precio y colección. El producto se crea activo por defecto.

---

## Criterios de Aceptación

### Escenario 1: Crear producto exitosamente
**Dado que** soy un administrador autenticado  
**Cuando** creo un producto con SKU único, nombre, precio > 0 y colección opcional  
**Entonces** el sistema crea el producto exitosamente  
**Y** `IsActive = true`  
**Y** se muestra confirmación

### Escenario 2: SKU duplicado
**Dado que** existe un producto con SKU "JOY-001"  
**Cuando** intento crear otro producto con el mismo SKU  
**Entonces** el sistema muestra error: "El SKU ya está en uso"  
**Y** no se crea el producto

### Escenario 3: Validación de campos requeridos
**Dado que** intento crear un producto sin completar campos obligatorios  
**Entonces** el sistema muestra mensajes de validación  
**Y** no se crea el producto

---

## Tareas

1. Backend: Endpoint `POST /api/products`
2. Frontend: Formulario de creación de producto
3. Validaciones: SKU único, precio > 0
