# HU-EP8-005: Configurar edición manual de precio por punto de venta

## Formato estándar

Como **administrador**, quiero **configurar si un punto de venta permite la edición manual de precios de venta** para **controlar qué ubicaciones pueden ajustar precios en el momento de la venta**.

---

## Descripción

El sistema debe permitir a los administradores activar o desactivar la opción `AllowManualPriceEdit` por cada punto de venta, tanto al crear como al editar. Cuando está habilitada, los operadores asignados a ese punto de venta podrán modificar el precio de venta al registrar una transacción. Por defecto, la opción está desactivada para todos los puntos de venta.

**Contexto de negocio:**
- Algunos puntos de venta (ej. tiendas propias) necesitan flexibilidad para ajustar precios en el momento de la venta (descuentos, negociaciones con clientes)
- Otros puntos de venta (ej. hoteles, terceros) deben usar siempre el precio oficial del catálogo
- La configuración es a nivel de punto de venta, no a nivel de producto ni de operador
- Solo los administradores pueden habilitar o deshabilitar esta opción

**Campo afectado:**
- `PointOfSale.AllowManualPriceEdit` (boolean, default `false`)

---

## Criterios de Aceptación

### Escenario 1: Crear punto de venta con edición de precio habilitada
**Dado que** soy un administrador autenticado
**Cuando** creo un nuevo punto de venta con `AllowManualPriceEdit = true`
**Entonces** el sistema crea el punto de venta con la opción habilitada
**Y** los operadores asignados podrán modificar precios al registrar ventas

### Escenario 2: Crear punto de venta sin especificar edición de precio
**Dado que** soy un administrador autenticado
**Cuando** creo un nuevo punto de venta sin modificar la opción de edición de precio
**Entonces** el sistema crea el punto de venta con `AllowManualPriceEdit = false` por defecto
**Y** los operadores usarán siempre el precio oficial del producto

### Escenario 3: Habilitar edición de precio en punto de venta existente
**Dado que** existe un punto de venta con `AllowManualPriceEdit = false`
**Cuando** edito el punto de venta y activo la opción "Permitir Edición Manual de Precio"
**Entonces** el sistema actualiza la configuración exitosamente
**Y** las nuevas ventas en ese punto de venta permitirán modificar el precio

### Escenario 4: Deshabilitar edición de precio en punto de venta existente
**Dado que** existe un punto de venta con `AllowManualPriceEdit = true`
**Cuando** edito el punto de venta y desactivo la opción
**Entonces** el sistema actualiza la configuración exitosamente
**Y** las ventas existentes con precio modificado no se ven afectadas
**Y** las nuevas ventas usarán el precio oficial del producto

### Escenario 5: Operador no puede modificar la configuración
**Dado que** soy un operador autenticado
**Cuando** intento modificar la configuración `AllowManualPriceEdit` de un punto de venta
**Entonces** el sistema rechaza la operación con 403 Forbidden
**Y** la configuración no se modifica

### Escenario 6: La configuración se incluye en las respuestas de API
**Dado que** consulto los puntos de venta (lista o detalle)
**Cuando** el sistema devuelve la información del punto de venta
**Entonces** incluye el campo `allowManualPriceEdit` en la respuesta
**Y** los formularios de venta pueden usar este valor para mostrar u ocultar el campo de precio editable
