# T-EP1-007-001: Create product search API endpoint

## Title
Create product search API endpoint

## Description

Implement GET /api/products/search endpoint for SKU exact match and name partial search functionality.

**Purpose:**
- Enable fast product lookup by SKU or name
- Support manual sales registration workflows
- Provide search results for product selection

**Details:**
- GET /api/products/search?query={term}
- Exact match for SKU field
- Partial match for Name field (case-insensitive)
- Return matching products with basic info
- Include primary photo in results
- Limit results for performance

## Acceptance Criteria

- [ ] SKU exact search working
- [ ] Name partial search implemented
- [ ] Case-insensitive search
- [ ] Performance optimized
- [ ] Results include primary photos

## Priority
High (essential for sales workflows)

## Estimation
2 SP
