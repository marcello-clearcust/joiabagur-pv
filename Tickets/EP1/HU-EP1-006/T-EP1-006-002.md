# T-EP1-006-002: Create product catalog page with data grid

## Title
Create product catalog page with data grid

## Description

Create product catalog page using Metronic data-grid component with pagination, product cards, and photo display.

**Purpose:**
- Display paginated product catalog
- Show product information and photos
- Enable navigation to product details
- Support large product inventories

**Details:**
- Metronic data-grid or card layout
- Product cards with primary photos
- Pagination controls
- SKU, name, price display
- Links to edit/view products
- Responsive design for mobile

## Acceptance Criteria

- [ ] Product catalog page created
- [ ] Data grid or card layout working
- [ ] Pagination implemented
- [ ] Product photos displayed
- [ ] Navigation to product management
- [ ] Mobile responsive

## Priority
High (catalog browsing UI)

## Estimation
4 SP

Frontend
