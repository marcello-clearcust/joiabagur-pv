# T-EP1-003-003: Integration tests for product update endpoint

## Title
Integration tests for product update endpoint

## Description

Create integration tests for PUT /api/products/{id} endpoint to verify product update functionality, SKU immutability, and validation rules.

**Purpose:**
- Test product update workflow end-to-end
- Verify SKU cannot be changed
- Ensure proper validation and business rules
- Test authentication and error handling

**Details:**
- Test successful product updates
- Test SKU immutability (attempted changes rejected)
- Test validation errors (invalid price, etc.)
- Test updating collection assignment
- Test product not found scenarios
- Verify UpdatedAt timestamp changes
- Test authentication requirements

**References:**
- Testing guide: `Documentos/Testing/Backend/03-testcontainers.md`
- User Story scenarios: All acceptance criteria in HU-EP1-003
- Stack: WebApplicationFactory, Testcontainers, PostgreSQL

- [ ] SKU immutability enforced and tested
- [ ] All validation scenarios tested
- [ ] Authentication and authorization tested

- [ ] Database state verification
