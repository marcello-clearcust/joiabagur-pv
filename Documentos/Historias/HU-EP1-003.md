# HU-EP1-003: Editar producto existente

## Formato estándar

Como **administrador**, quiero **editar información de productos existentes** para **actualizar precios, descripciones u otros datos cuando cambian**.

---

## Descripción

Permite modificar nombre, descripción, precio y colección de productos existentes. El SKU es inmutable.

---

## Criterios de Aceptación

### Escenario 1: Editar producto exitosamente
**Dado que** existe un producto en el sistema  
**Cuando** edito su nombre, descripción, precio o colección  
**Entonces** el sistema actualiza el producto  
**Y** se actualiza `UpdatedAt`

### Escenario 2: SKU inmutable
**Dado que** estoy editando un producto  
**Cuando** intento cambiar el SKU  
**Entonces** el campo SKU está deshabilitado  
**Y** no se puede modificar

---

## Tareas

1. Backend: Endpoint `PUT /api/products/{id}`
2. Frontend: Formulario de edición pre-poblado
3. Validaciones: SKU inmutable, precio > 0
