# T-EP1-004-002: Implement product photo management service

## Title
Implement product photo management service

## Description

Create ProductPhotoService to handle photo-specific business logic including upload, validation, primary photo management, and display ordering.

**Purpose:**
- Centralize photo management business logic
- Handle primary photo designation
- Manage display order for photo galleries
- Validate photo operations against business rules

**Details:**
- IProductPhotoService interface
- ProductPhotoService implementation
- Photo upload with automatic ordering
- Primary photo designation logic
- Display order management
- Validation for photo limits and formats
- Integration with IFileStorageService

**References:**
- User Story: `Documentos/Historias/HU-EP1-004.md`
- Domain entities: ProductPhoto entity
- File storage: IFileStorageService
