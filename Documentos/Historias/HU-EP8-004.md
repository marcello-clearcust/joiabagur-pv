# HU-EP8-004: Visualizar puntos de venta disponibles

## Formato estándar

Como **administrador u operador**, quiero **visualizar la lista de puntos de venta disponibles** para **seleccionar el punto de venta correcto al realizar operaciones**.

---

## Descripción

El sistema debe mostrar una lista de puntos de venta según el rol del usuario. Los administradores ven todos los puntos de venta con indicadores visuales para el estado de activación, mientras que los operadores solo ven sus puntos de venta asignados.

---

## Criterios de Aceptación

### Escenario 1: Administrador ve todos los puntos de venta con indicadores de estado
**Dado que** soy un administrador autenticado
**Cuando** visualizo la lista de puntos de venta
**Entonces** veo todos los puntos de venta (activos e inactivos)
**Y** los puntos de venta inactivos se muestran con indicadores visuales (deshabilitados, atenuados)
**Y** puedo ver información básica (nombre, código, estado de activación)

### Escenario 2: Operador ve solo puntos de venta asignados
**Dado que** soy un operador asignado a puntos de venta A y B  
**Cuando** visualizo la lista de puntos de venta  
**Entonces** solo veo los puntos de venta A y B  
**Y** no veo otros puntos de venta

---

## Tareas

1. Backend: Endpoint `GET /api/point-of-sales` que retorna todos los puntos de venta
2. Frontend: Lista de puntos de venta con indicadores visuales para estado de activación
3. Control de acceso: Filtrar según asignaciones para operadores (solo UI)
4. UI: Mostrar puntos de venta inactivos con estilo visual diferenciado (atenuado, deshabilitado)
