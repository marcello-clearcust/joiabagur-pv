# T-EP1-003-002: Create product edit form page

## Title
Create product edit form page

## Description

Create React page for editing existing products with pre-populated form fields, SKU field disabled, and proper validation feedback.

**Purpose:**
- Provide interface for updating product information
- Pre-populate form with existing product data
- Disable SKU field to prevent changes
- Show clear feedback on successful updates

**Details:**
- Load existing product data on page load
- Pre-populate all form fields
- Disable SKU field with visual indication
- Allow editing of Name, Description, Price, Collection
- Real-time validation with error messages
- Update button with loading state
- Success feedback and navigation options
- Handle product not found scenarios

**References:**
- User Story: `Documentos/Historias/HU-EP1-003.md` (SKU immutability requirement)
- Metronic components: Reuse form components from creation
- Frontend patterns: Check existing edit forms in codebase

## Acceptance Criteria

- [ ] Product edit form page created
- [ ] Existing product data loaded and displayed
- [ ] SKU field disabled and visually indicated
- [ ] Form validation for editable fields
- [ ] Successful update feedback
- [ ] Error handling for not found products
- [ ] Proper loading states
