# T-EP1-004-003: Create product photo upload UI component

## Title
Create product photo upload UI component

## Description

Create photo upload component using Metronic file-upload with drag-and-drop, preview, and validation for product photo management.

**Purpose:**
- Enable photo upload in product forms
- Provide visual feedback and validation
- Support multiple photo uploads
- Integrate with existing product forms

**Details:**
- Metronic file-upload component integration
- Drag-and-drop upload area
- Photo preview thumbnails
- Format and size validation
- Progress indicators
- Error feedback display

## Acceptance Criteria

- [ ] Photo upload component working
- [ ] Drag-and-drop functionality
- [ ] File validation feedback
- [ ] Preview thumbnails shown
- [ ] Integration with product forms

High (photo upload UI)
