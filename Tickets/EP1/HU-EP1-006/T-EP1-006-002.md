# T-EP1-006-002: Create product catalog page with data grid

## Title
Create product catalog page with data grid and quantity indicators

## Description

Create product catalog page using Metronic data-grid component with pagination, product cards, photo display, and quantity indicators. **Backend filtering ensures operators see only products assigned to their POS inventories.**

**Purpose:**
- Display paginated product catalog (filtered by backend based on role)
- Show product information, photos, and quantity status
- Enable navigation to product details
- Support large product inventories
- Display warnings for products with zero inventory

**Details:**
- Metronic data-grid or card layout
- Product cards with primary photos and quantity badges
- Display "Sin stock" indicator for products with qty=0
- Pagination controls
- SKU, name, price, quantity display
- Links to edit/view products (role-aware)
- Responsive design for mobile
- Empty state for operators with no assigned products

## Acceptance Criteria

- [ ] Product catalog page created with role-aware display
- [ ] Data grid or card layout working
- [ ] Quantity indicators displayed on product cards
- [ ] "Sin stock" warning badge shown for products with qty=0
- [ ] Pagination implemented
- [ ] Product photos displayed
- [ ] Navigation to product management (edit for admin, view for operator)
- [ ] Empty state message for operators with no assigned products
- [ ] Mobile responsive
- [ ] Backend handles all filtering (frontend displays filtered results)
