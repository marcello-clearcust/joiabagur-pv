# T-EP1-007-002: Add product search functionality to catalog

## Title
Add product search functionality to catalog with quantity indicators

## Description

Integrate search input in product catalog with real-time search by SKU and name, displaying filtered results with quantity indicators. **Backend handles role-based filtering automatically.**

**Purpose:**
- Enable fast product lookup (results filtered by backend based on role)
- Support sales workflows
- Provide search within catalog view
- Real-time search feedback
- Display quantity warnings for products with zero inventory

**Details:**
- Search input in catalog header
- Real-time search as user types (300ms debounce)
- Display quantity indicators on search results
- Show "Sin stock" warning badge for qty=0 products
- SKU exact match priority in display
- Name partial match fallback
- Clear search results display
- Search state management
- Backend handles all access control and filtering

## Acceptance Criteria

- [ ] Search input integrated in catalog header
- [ ] Real-time search working with 300ms debounce
- [ ] SKU exact match results prioritized in display
- [ ] Name partial search working
- [ ] Quantity indicators displayed on search results
- [ ] "Sin stock" warning badge shown for products with qty=0
- [ ] Clear search state management
- [ ] Search results properly displayed (backend-filtered)
- [ ] Clear search button working
- [ ] "No results found" message for empty results
- [ ] Search loading indicator visible during search
