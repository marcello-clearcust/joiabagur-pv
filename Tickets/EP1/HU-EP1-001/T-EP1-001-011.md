# T-EP1-001-011: Create product import page with file upload UI

## Title
Create product import page with file upload UI

## Description

Create React page for product import using Metronic components (file-upload, form, dialog) with Excel file upload, preview, validation feedback, and import progress indication.

**Purpose:**
- Provide user-friendly interface for bulk product import
- Show import preview and validation errors before confirmation
- Enable download of Excel template for users
- Display import results with success/error details

**Details:**
- Use Metronic `file-upload` component for Excel file selection
- Implement drag-and-drop upload area
- Show file preview with row count and basic validation
- Display validation errors in a clear, actionable format
- Allow download of Excel template with required columns
- Show progress indicator during import processing
- Display detailed results (created, updated, errors) after import
- Use Metronic `dialog` for confirmation and results display
