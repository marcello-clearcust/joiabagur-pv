# HU-EP5-002: Buscar venta original para asociar devolución

## Formato estándar

Como **administrador**, quiero **buscar la venta original para asociar una devolución** para **mantener la trazabilidad entre ventas y devoluciones**.

---

## Descripción

Permite buscar ventas por SKU, fecha, punto de venta u operador para encontrar la venta original a la que asociar la devolución.

---

## Criterios de Aceptación

### Escenario 1: Buscar venta por filtros
**Dado que** existen ventas registradas  
**Cuando** busco ventas con filtros (SKU, fecha, punto de venta)  
**Entonces** veo lista de ventas que cumplen los criterios  
**Y** puedo seleccionar una venta para asociar la devolución

### Escenario 2: Visualizar detalles de venta
**Dado que** encuentro una venta en la búsqueda  
**Cuando** visualizo sus detalles  
**Entonces** veo SKU, producto, precio, fecha, operador, método de pago y foto (si existe)

---

## Tareas

1. Backend: Endpoint `GET /api/sales` con filtros para búsqueda
2. Frontend: Búsqueda de ventas con filtros y selección
