# T-EP1-001-004: Implement Product repository classes

## Title
Implement Product repository classes

## Description

Implement concrete repository classes (ProductRepository, ProductPhotoRepository, CollectionRepository) that provide data access for Product domain entities using Entity Framework Core.

**Purpose:**
- Provide concrete implementations of repository interfaces
- Enable CRUD operations and specialized queries for products
- Support efficient data access patterns for product management

**Details:**
- ProductRepository: Implement IProductRepository with EF Core queries
- ProductPhotoRepository: Implement IProductPhotoRepository with photo operations
- CollectionRepository: Implement ICollectionRepository for collections
- Include eager loading for related entities (photos, collections)
- Implement pagination support for large datasets
- Follow existing repository implementation patterns (UserRepository, etc.)
- Include proper error handling and logging

**References:**
- Repository interfaces: IProductRepository, IProductPhotoRepository, ICollectionRepository
- Existing implementations: Check UserRepository, PointOfSaleRepository
- EF Core patterns: Check existing query patterns in codebase

## Acceptance Criteria

- [ ] ProductRepository implements all IProductRepository methods
- [ ] ProductPhotoRepository implements photo-specific operations
- [ ] CollectionRepository implements collection CRUD operations
- [ ] EF Core queries optimized with proper includes and filtering
- [ ] Pagination implemented for list operations
- [ ] Error handling and logging included
- [ ] Unit tests verify repository operations work correctly

## Priority
High (enables data access for all product features)

5 SP
