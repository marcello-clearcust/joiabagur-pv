# HU-EP8-003: Activar/desactivar punto de venta

## Formato estándar

Como **administrador**, quiero **activar o desactivar puntos de venta** para **controlar qué puntos de venta están disponibles para operaciones sin eliminarlos del sistema**.

---

## Descripción

El sistema debe permitir activar o desactivar puntos de venta. Los puntos de venta desactivados no deben aparecer en listas para nuevas operaciones pero mantienen su historial.

---

## Criterios de Aceptación

### Escenario 1: Desactivar punto de venta
**Dado que** existe un punto de venta activo  
**Cuando** lo desactivo  
**Entonces** `IsActive = false`  
**Y** no aparece en listas para nuevas ventas o asignaciones  
**Y** se mantiene el historial

### Escenario 2: Activar punto de venta
**Dado que** existe un punto de venta inactivo  
**Cuando** lo activo  
**Entonces** `IsActive = true`  
**Y** vuelve a estar disponible para operaciones

---

## Tareas

1. Backend: Endpoint para toggle de `IsActive`
2. Frontend: Toggle o botón de activar/desactivar
3. Validaciones: Prevenir desactivar si tiene operaciones pendientes (opcional)
