# T-EP1-002-004: Frontend tests for product creation form

## Title
Frontend tests for product creation form

## Description

Create component tests for the product creation form using React Testing Library and MSW to test form interactions, validation, and API integration.

**Purpose:**
- Ensure form works correctly with all validation scenarios
- Test user interactions and feedback
- Verify API integration and error handling

**Details:**
- Test form field interactions and validation
- Test collection dropdown population
- Test form submission success and error scenarios
- Test SKU uniqueness validation
- Test loading states and user feedback
- MSW mocking for API responses
- Use accessible queries throughout

**References:**
- Testing guide: `Documentos/Testing/Frontend/03-tests-componentes.md`
- API mocking: `Documentos/Testing/Frontend/04-mocking-api.md`
- User Story scenarios: All acceptance criteria in HU-EP1-002
- Stack: Vitest, React Testing Library, MSW

## Acceptance Criteria

- [ ] Form validation tested for all fields
- [ ] Collection dropdown tested
- [ ] Form submission tested (success/error)
- [ ] API error handling tested
- [ ] Loading states tested
- [ ] Accessible queries used
- [ ] MSW mocks API correctly
